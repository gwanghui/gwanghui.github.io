Linux Run Level

런레벨 상태
0 시스템 중지
1 단일 사용자 모드(복구 모드)
2 NFS 없는 다중 사용자 모드
3 완전한 다중 사용자 모드
4 미사용
5 그래픽 인터페이스 모드
6 시스템 재시작
  • Debian, Ubuntu : Run Level 2
  • RedHat, fedora, CentOS, gentoo : Run Level 3
