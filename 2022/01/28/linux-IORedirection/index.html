IO Redirection

three default file

  • stdin: the keyboard
  • stdout: the screen
  • stderr: error messages output to the screen

assigned a file descriptor

  • stdin: 0
  • stdout: 1
  • stderr: 2

Closing File Descriptors

  • Close input file descriptor n. : n<&-
  • Close stdin. : 0<&-, <&-
  • Close output file descriptor n : n>&-
  • Close stdout :1>&-, >&-
